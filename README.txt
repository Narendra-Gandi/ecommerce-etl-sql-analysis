# 🛒 E-Commerce ETL & SQL Analysis Project

## 🚀 Overview

This project demonstrates an **end-to-end data pipeline** for an e-commerce platform. It involves extracting raw data from multiple sources, transforming it into a clean and structured format, loading it into a relational database, and generating business insights using SQL.

The goal is to simulate a real-world data engineering and data analyst workflow.

---

## 🎯 Objectives

* Build a complete **ETL pipeline using Python**
* Design a **relational database in MySQL**
* Ensure **data quality and integrity**
* Generate **business insights using SQL queries**

---

## 🧰 Tech Stack

* **Python** (Pandas, NumPy)
* **MySQL**
* **Jupyter Notebook**
* **SQL (Joins, Aggregations, Constraints)**
* **CSV & Excel Files**

## ⚙️ ETL Pipeline

### 🔹 Extract

* Imported data from CSV and Excel files using Pandas
* Combined multiple datasets into structured DataFrames

### 🔹 Transform

* Cleaned invalid emails and missing values
* Converted data types (dates, numeric fields)
* Standardized formats for consistency
* Maintained relationships between datasets

### 🔹 Load

* Loaded cleaned data into MySQL database
* Applied primary keys and foreign key relationships
* Ensured referential integrity

---

## 🧪 Data Validation

* Verified row counts between source files and database
* Applied constraints like `CHECK`, `NOT NULL`
* Removed invalid and inconsistent records

---

## 📊 Key Business Insights

### 🔹 Customer Lifetime Value (CLV)

* Calculated total revenue generated by each customer
* Identified high-value customers

### 🔹 Product Inventory Analysis

* Monitored stock levels and product availability
* Identified low-stock and high-demand products

---

## 📌 Sample SQL Query

```sql
SELECT UserID, SUM(TotalAmount) AS CLV
FROM Orders
GROUP BY UserID
ORDER BY CLV DESC;
```
